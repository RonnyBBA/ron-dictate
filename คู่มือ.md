# 🎙️ Ron Dictate — พูดแล้วพิมพ์ ในเครื่องคุณเอง

> by **Ronny BBA** · ฟรี · ไม่จำกัดคำ · ไม่มีรายเดือน · เสียงไม่ออกจากเครื่องคุณเลย

พูดภาษาไทย (ปนอังกฤษได้) แล้วข้อความพิมพ์ลงตรงเคอร์เซอร์ในแอปไหนก็ได้ — Line, เว็บ, Notes, Word, โปรแกรมตัดต่อ ใช้โมเดล Whisper ตัวเดียวกับแอป dictation ที่เก็บรายเดือน แต่รันในเครื่องคุณ 100%

**ต้องใช้:** Mac (ชิป M1-M4 = เร็วสุด · Intel ก็ได้แต่ช้ากว่า) · macOS 13 ขึ้นไป · เนื้อที่ว่าง ~4GB · เน็ต (เฉพาะตอนติดตั้ง)

---

## ติดตั้ง (ครั้งเดียว ~5-10 นาที)

### ขั้น 1 — ดับเบิลคลิก `ติดตั้ง.command`
- ⚠️ **ครั้งแรก macOS อาจฟ้องว่า "ไม่สามารถเปิดได้"** → ให้**คลิกขวาที่ไฟล์ → เลือก "เปิด" (Open) → กด "เปิด" ยืนยัน** (ทำครั้งเดียว)
- หน้าต่างดำๆ จะขึ้น ปล่อยให้มันทำงานจนขึ้น **"🎉 ติดตั้งเสร็จ!"** (โหลดโมเดล ~1.6GB รอหน่อย)

### ขั้น 2 — กด "อนุญาต" ให้ครบ 3 จอ
ระบบจะเด้งจอถามทีละอัน (บางจอพาไปหน้า การตั้งค่าระบบ ให้เปิดสวิตช์ **เทอร์มินัล / Terminal หรือ Python**):

| จอที่เด้ง | ถามเรื่อง | เพื่ออะไร |
|---|---|---|
| 🎤 ไมโครโฟน | เข้าถึงไมค์ | อัดเสียงพูดของคุณ |
| ⌨️ การตรวจสอบอินพุต (Input Monitoring) | รับการกดแป้นพิมพ์ | ให้รู้ว่าคุณจิ้มปุ่ม ⌥ |
| ♿ การช่วยการเข้าถึง (Accessibility) | ควบคุมเครื่อง | วางข้อความแทนคุณ (Cmd+V) |

ถ้าจอไหนไม่เด้ง: การตั้งค่าระบบ → **ความเป็นส่วนตัวและความปลอดภัย** → หาหมวดตามตาราง → เปิดสวิตช์ให้ **เทอร์มินัล**

### ขั้น 3 — ดับเบิลคลิก `เปิดใช้งาน.command` อีกรอบ
(permission จะติดผลหลังเปิดใหม่ 1 ครั้ง) → รอไอคอนเมนูบาร์เปลี่ยนจาก ⏳ เป็น **💤 = พร้อมใช้**

---

## วิธีใช้ (3 จังหวะ)

1. วางเคอร์เซอร์ในช่องพิมพ์ที่ไหนก็ได้
2. **จิ้ม ⌥ Option ซ้าย 1 ที แล้วปล่อยมือ** → เสียง "ป๊อป" + ไอคอนเป็น 🎙️ → พูดยาวแค่ไหนก็ได้
3. พูดจบ **จิ้ม ⌥ ซ้ายอีก 1 ที** → รอ 1-3 วิ → ข้อความพิมพ์ลงให้ทั้งก้อน + เสียง "กริ๊ง"

ไอคอนเมนูบาร์: 💤 พร้อมใช้ · 🎙️ กำลังอัด · ⏳ กำลังโหลด/ถอดเสียง · 🚫 พักอยู่
(ใช้ ⌥ ประกอบคีย์ลัดอื่นได้ปกติ เช่น ⌥+ลูกศร — ระบบนับเฉพาะ "จิ้มเดี่ยว")

**เคล็ดลับ:** ถ้าข้อความไม่ยอมพิมพ์ลงช่อง ข้อความล่าสุดอยู่ใน clipboard เสมอ → กด **Cmd+V** วางเองได้เลย

---

## ปรับแต่ง (ไฟล์ `config.json` เปิดแก้ด้วย TextEdit ได้)

- **ศัพท์เฉพาะของคุณ** (ชื่อแบรนด์ ชื่อคน คำอังกฤษ) → เติมใน `"vocab"` — โมเดลจะถอดคำพวกนี้ถูกขึ้นมาก
- **คำที่ถอดเพี้ยนซ้ำๆ** → เติมคู่ผิด→ถูก ใน `"corrections"` เช่น `"คลอดโค้ด": "Claude Code"`
- **เปลี่ยนปุ่ม** → `"hotkey"`: `alt_l` (⌥ ซ้าย) / `alt_r` (⌥ ขวา)
- **โมเดล** → คลิกไอคอนเมนูบาร์: turbo (แนะนำ) / large-v3 (แม่นสุดแต่ช้ากว่ามาก)
- แก้ config แล้ว ดับเบิลคลิก `เปิดใช้งาน.command` ใหม่ 1 รอบ

## แก้ปัญหายอดฮิต

| อาการ | วิธีแก้ |
|---|---|
| จิ้ม ⌥ แล้วเงียบ ไม่มีเสียงป๊อป | เปิดสวิตช์ "การตรวจสอบอินพุต" ให้เทอร์มินัล → เปิดแอปใหม่ |
| ป๊อปมี แต่ข้อความไม่พิมพ์ | เปิดสวิตช์ "การช่วยการเข้าถึง" → เปิดแอปใหม่ · ระหว่างนี้ Cmd+V วางเองได้ |
| ขึ้น "ไม่ได้ยินเสียงพูด" | อนุญาตไมโครโฟน / พูดใกล้ไมค์ขึ้น / อย่าจิ้มติดกันเร็วเกิน |
| คำเฉพาะถอดเพี้ยน | เติมคำใน `vocab` + `corrections` (ดูหัวข้อปรับแต่ง) |
| ไอคอนค้าง ⏳ นาน | ครั้งแรกโหลดโมเดล ~20 วิ ปกติ · ถ้าเกิน 2 นาที เปิด `dictate.log` ดูบรรทัดล่าสุด |
